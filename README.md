# BMO Care - نظام تعليمي تفاعلي صوتي للأطفال 🤖

## نظرة عامة
BMO Care هو تطبيق تعليمي تفاعلي صوتي مصمم خصيصاً للأطفال من عمر 3-12 سنة، مبني على Blazor Server .NET 10. يستخدم التطبيق تقنيات Web Speech API للتعرف على الصوت والنطق الصوتي.

## المميزات الرئيسية ✨

### 1. **تفاعل صوتي كامل**
- التواصل مع الطفل بشكل كامل عبر الصوت (TTS + Speech Recognition)
- لا حاجة لكتابة أي نص من الطفل
- واجهة بسيطة جداً مع الحد الأدنى من اللمس

### 2. **أقسام تعليمية متنوعة**
- **الحروف الإنجليزية** (A-Z)
- **الحروف العربية** (أ-ي)
- **الأرقام** (1-20)
- **محادثة حرة** (قادمة قريباً)

### 3. **نظام تقييم ذكي**
- تقييم دقيق لإجابات الأطفال باستخدام:
  - Confidence scoring
  - Levenshtein distance للمقارنة اللفظية
  - قبول نطق متعدد لنفس الحرف/الرقم
- Thresholds قابلة للتعديل حسب عمر الطفل

### 4. **لوحة تحكم لولي الأمر**
- عرض إحصائيات شاملة للطفل
- تتبع التقدم والأداء
- تصدير البيانات
- إمكانية مسح البيانات

### 5. **وجه BMO التفاعلي**
- وجه متحرك يتفاعل مع الصوت
- تعبيرات مختلفة (سعيد، حزين، متكلم)
- تزامن حركة الفم مع النطق

## سير العمل 🎯

### 1. شاشة الترحيب (0-5 ثواني)
- يظهر وجه BMO
- تحية صوتية: "أهلاً! أنا BMO، صديقك الجديد..."

### 2. إدخال الاسم (صوتي)
- BMO يسأل: "ممكن أعرف اسمك؟"
- الطفل ينطق اسمه
- النظام يستخرج ويخزن الاسم

### 3. اختيار القسم (لمس)
- 4 أزرار كبيرة: إنجليزي، عربي، رياضة، نتكلم
- الطفل يلمس الاختيار
- تأكيد صوتي

### 4. جلسة التعلم (صوتي بالكامل)
- عرض الحرف/الرقم (3 ثواني)
- BMO ينطق الحرف
- الطفل يكرر
- تقييم فوري وتغذية راجعة

### 5. لوحة ولي الأمر
- الوصول عبر زر "ولي الأمر" أو أمر صوتي
- عرض إحصائيات مفصلة

## الإعدادات والتخصيص ⚙️

### تعديل thresholds حسب العمر
في `appsettings.json`:
```json
"ConfidenceThresholds": {
  "Age3to5": 0.28,
  "Age6to8": 0.35,
  "Age9to12": 0.45
}
```

### تعديل النطق المقبول
في `AssessmentService.cs`، عدّل دالة `GetEnglishPronunciations()`:
```csharp
'A' => new List<string> { "a", "ay", "eh", "ei", "ا", "إيه" }
```

### تعديل الأصوات
في `bmokids.js`، عدّل إعدادات TTS:
```javascript
utterance.rate = 0.95;  // سرعة النطق
utterance.pitch = 1.0;  // طبقة الصوت
```

## المشاكل الشائعة وحلولها 🔧

### 1. Speech Recognition لا يعمل
**الحل:**
- استخدم Chrome أو Edge
- تأكد من HTTPS
- امنح إذن المايكروفون

### 2. الصوت لا يعمل
**الحل:**
- تأكد من تشغيل الصوت في المتصفح
- تحقق من إعدادات النظام
- جرب متصفح مختلف

### 3. دقة التعرف منخفضة
**الحل:**
- خفّض threshold في الإعدادات
- تأكد من بيئة هادئة
- استخدم مايكروفون جيد
